# Rick and Morty iOS App

Una aplicación nativa desarrollada en Swift que consume la API de Rick and Morty.

## Características principales

* **Exploración de Personajes:** Listado dinámico con consumo de API REST.
* **Paginación Infinita:** Carga automática de resultados conforme el usuario hace scroll.
* **Buscador en Tiempo Real:** Filtrado de personajes por nombre directamente desde la API.
* **Sistema de Favoritos:** Persistencia local mediante Core Data. Permite añadir/eliminar personajes.

## Tecnologías utilizadas

* **Lenguaje:** Swift 5.10
* **Frameworks de UI:** UIKit y SwiftUI.
* **Networking:** Alamofire.
* **Persistencia:** Core Data.
* **Arquitectura:** MVC (Model-View-Controller).

## Instalación

1. Clona el repositorio:
   ```bash
   git clone [https://github.com/tu-usuario/RickAndMortyAPI.git](https://github.com/tu-usuario/RickAndMortyAPI.git)
   
2. Abre RickAndMortyAPI.xcodeproj en Xcode.

3. El proyecto gestiona dependencias mediante Swift Package Manager. Xcode las descargará automáticamente (Alamofire).

4. Ejecuta en el simulador de tu preferencia.
